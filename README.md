- **Routing:** React Router
- **Internationalization:** i18next (EN/ES/CA)

## 🚀 Local Development

Want to run this portfolio locally?

```bash
# Clone the repository
git clone https://github.com/jcatalaamat/astralamat.git
cd astralamat

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
